## I build production systems with sharp edges made legible.

Senior Fullstack Engineer based in Belgrade, with 7+ years across Web3, applied AI, fintech, and enterprise platforms. I like work where the interface is not decoration: it is how people inspect complex state, trust automation, and make better decisions.

My current center of gravity:

- **Auditable AI systems:** agent workflows, structured outputs, evidence maps, evals, trace logs, replayable runs.
- **Web3 infrastructure:** Cosmos SDK products, chain indexers, block explorers, DAO frameworks, wallet signing flows.
- **Data-heavy product surfaces:** React, Next.js, TanStack, TypeScript, PostgreSQL, real-time UI, design systems, tests.

## Working style

- I prefer systems that leave artifacts behind: traces, contracts, schemas, reports, and reviewable state.
- I enjoy the full stack when the boundaries matter: data model, API, product workflow, interface, tests, deploy path.
- I care about design as operational clarity. The best UI makes hard things easier to inspect.
- I ship in loops: spec, build, verify, tighten, then make the result easier for the next person to operate.
